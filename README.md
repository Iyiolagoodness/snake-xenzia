# 🐍 Snake Studio

A feature-packed Snake game built with React + Canvas.

## Quick Start
```bash
npm install
npm start
```

## Controls
- **WASD** or **Arrow Keys** — move
- **P** or **Escape** — pause

## Features
- 🎮 Classic & ⏱ Time Attack modes
- 🟢🟡🔴 3 difficulty levels (speed increases every 5 foods)
- 🍎🍇💣⏱ 4 food types with weighted spawning
- ⚡🐌⭐🛡 4 power-ups
- 🌀 Portal teleporters
- 🧱 Obstacles (more per level)
- 🐍 4 unlockable skins (Classic / Neon / Rainbow / Fire)
- 🏆 Combo system (eat 3 fast → bonus points)
- 🎖 Achievement pop-ups
- 🔊 Web Audio sound effects
- 💾 High score saved in localStorage
- ✨ Particle explosions on eat
- 👁 Snake eyes that track direction
